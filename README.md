# Indian Budget Analysis (1947–2026) | SQL Data Analysis Project

## 📌 Project Overview

This project analyzes India's Union Budget from **1947 to 2026** using **MySQL**. The objective is to explore historical budget trends, identify spending patterns, compare yearly growth, and answer business-oriented analytical questions using SQL.

The project demonstrates both **fundamental** and **intermediate SQL** concepts commonly used in data analysis.

---

## 🎯 Objectives

- Explore and understand the Indian Budget dataset.
- Perform descriptive and statistical analysis.
- Analyze budget trends over multiple decades.
- Calculate year-over-year budget growth.
- Rank budget allocations using SQL window functions.
- Generate business insights from historical budget data.

---

## 🛠️ Tools Used

- MySQL Workbench
- SQL

---

## 📂 Dataset

**Dataset:** Indian Union Budget (1947–2026)

The dataset contains yearly budget information along with allocations for various government sectors.

# 📊 Analysis Performed

The project answers 50 business-oriented SQL questions, including:

- Display dataset overview
- Find total number of budget records
- Identify earliest and latest budget year
- Calculate total budget allocation
- Calculate average budget
- Find highest and lowest budgets
- Identify years with maximum and minimum budget
- Analyze Education, Health and Defence expenditure
- Compare budgets before and after economic reforms
- Find top budget years
- Calculate previous and next year's budget
- Calculate running total
- Calculate year-over-year budget growth
- Rank budgets using SQL window functions
- Categorize budgets using CASE statements
- Analyze highest growth years using CTEs
- Compare budgets against overall averages
- Generate a comprehensive budget summary

---

# 📈 Key Insights

- The dataset covers nearly eight decades of India's budget history.
- Government expenditure has shown a significant upward trend over time.
- Recent years record substantially higher budget allocations than earlier decades.
- Running total analysis highlights the cumulative growth in government expenditure.
- Year-over-year growth analysis identifies periods of rapid budget expansion.
- Ranking functions help identify the highest and lowest budget years.
- CASE statements simplify budget classification into meaningful categories.
- CTEs improve readability while solving complex analytical problems.
